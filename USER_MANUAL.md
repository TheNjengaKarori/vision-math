# Comprehensive User's Manual

Welcome to the accessible math grid tool! This tool is designed to provide an easy-to-use, inclusive, and keyboard-friendly environment for writing math equations, working in collaborative learning groups, performing quick calculations, and jotting down rough notes—all aided by a built-in voice assistant.

This manual explains all the features and capabilities of the tool, simply and clearly, without technical jargon. 

---

## 1. Getting Started: The Spatial Math Grid

The main screen is a **Spatial Math Grid**—a canvas divided into a grid similar to a spreadsheet. It allows you to align math problems, create equations step-by-step, and easily read calculations line by line.

*   **Typing Math:** You can type directly into the grid. Special words are automatically converted into math symbols. For example, typing "pi" will turn into "π", and "sq", "qr", or "sqrt" will become the square root symbol "√".
*   **Moving Around:** Use the **Arrow Keys (Up, Down, Left, Right)** on your keyboard to navigate directly between the grid cells.
*   **Evaluation:** Once you have typed a complete math problem on a row (like "2 + 2"), press **Ctrl (or Cmd) + =** to evaluate it. The result will instantly appear next to your equation.

### Helpful Grid Shortcuts:
*   **Context Locator:** If you ever get lost on the grid, quickly double-press the **Shift** key. The voice assistant will tell you exactly where you are!
*   **Undo/Redo:** Made a mistake? Press **Ctrl / Cmd + Z** to undo, and **Ctrl / Cmd + Y** to redo.

---

## 2. Voice Reader (Audio Engine)

To make the app completely accessible, it features a built-in voice reader that speaks back to you. The reader can announce the mathematical expression as you type, state your location on the grid, and read the results of calculations.

*   **Muting / Unmuting:** By default, the voice reader is muted. To turn the voice on or off, simply tap the **Ctrl or Cmd** key once quickly.

---

## 3. The Floating Calculator

You don't need to leave the app to do a quick calculation. The tool comes with a built-in floating calculator that speaks the numbers as you interact with them!

*   **How to Open:** Press **Alt + C** (or Option + C on Mac). You can also launch it from the Command Palette or the Help menu.
*   **Two Display Modes:** 
    *   **Standard View:** A classic calculator view for simple arithmetic. To switch to this mode, press **Ctrl (or Cmd) + 1**.
    *   **Scientific View:** Expanded functionality including Trigonometry (sin, cos, tan), log, exponents, and more! To switch to this mode, press **Ctrl (or Cmd) + 2**.
*   **Close the Calculator:** Press the **Esc** key.

---

## 4. The Scratchpad (Rough Work)

If you need a blank space to jot down notes or do some rough working, the Scratchpad gets the job done without cluttering your main grid.

*   **How to Open:** Press **Alt + R** (or Option + R on Mac), or simply type `rrr` quickly anywhere on the grid.
*   **How to Use:** The scratchpad is a large, open text document. Type whatever notes you like. It covers the grid until you close it by toggling it off using the exact same shortcut used to open it.

---

## 5. Collaboration and Classroom Tools

Working in a group or a classroom setting? The tool gives you smart features to participate effectively:

*   **Raise your Hand:** If you need help from a teacher or guide, type the letter `H` three times quickly (`HHH`) anywhere. The app will visually and audibly assert that you have raised your hand.
*   **Broadcast Line:** Type the letter `B` three times quickly (`BBB`) to "broadcast" your current equation line to others in your group or classroom.
*   **Pass Control:** If you are sharing a session and want to pass control to another user, press **Ctrl (or Cmd) + P**.
*   **Focus Navbar:** Need to access group settings? Press **Alt + T** (or Option + T) to focus the Navigation Bar at the top of the screen. You can then use your arrow keys to move through the top bar menus.

---

## 6. Command Palette

The Command Palette is like a universal search engine for everything your tool can do. Instead of remembering all the shortcuts, you can just open the palette and search for what you want!

*   **How to Open:** Press **Ctrl (or Cmd) + M**.
*   **How to Use:** Start typing the action you want to do (e.g., "mute", "calculator", "fraction"). Use your Up and Down arrow keys to find it from the list, then press **Enter** to execute that action.

---

## 7. Complete List of Keyboard Shortcuts

Here is your quick-reference sheet for all the keyboard shortcuts mentioned in this manual:

### General Actions:
*   **Shift + ?:** Open the complete Help Menu.
*   **Ctrl / Cmd (Tap):** Toggle Audio / Voice Reader on or off.
*   **Ctrl / Cmd + M:** Open the Command Palette to search for features.
*   **Alt / Option + T:** Focus the Top Navigation Bar.
*   **Double-click Shift:** Context Locator ("Where Am I?").

### Math & Grid:
*   **Ctrl / Cmd + =:** Evaluate the math equation on the current row.
*   **Ctrl / Cmd + Z:** Undo a change.
*   **Ctrl / Cmd + Y:** Redo a change.
*   **Alt / Option + R:** Toggle the Scratchpad (or type `rrr`).
*   **Ctrl / Cmd + S:** Submit your work.
*   **Ctrl / Cmd + R:** Reset the grid for the next user.

### Calculator:
*   **Alt / Option + C:** Open the Floating Voice-Assisted Calculator.
*   **Ctrl / Cmd + 1:** Switch to Standard Calculator View.
*   **Ctrl / Cmd + 2:** Switch to Scientific Calculator View.
*   **Esc:** Close Calculator (or close any open menu).

### Classroom & Collaboration:
*   **H H H (Typed quickly):** Raise Hand to get help in queue.
*   **B B B (Typed quickly):** Broadcast your current line.
*   **Ctrl / Cmd + P:** Pass Control to another user in your group.
*   **Ctrl / Cmd + G:** Start or toggle Group Work mode.

---

## 8. Automatically Recognized Math Words (Auto-completes)

The tool recognizes standard math wording and Nemeth braille codes. As soon as you type them in the grid, it converts them automatically:

*   **pi:** π
*   **sin, cos, tan:** Trigonometry symbols
*   **log:** Logarithm
*   **sq, qr, sqrt:** Square Root ( √ )
*   **alpha, beta, theta:** Greek symbols ( α, β, θ )
*   **inf:** Infinity ( ∞ )
*   **sum:** Summation ( ∑ )
*   **int:** Integral ( ∫ )

*Nemeth Code users can also type ASCII codes like `.p` for Pi, or `,i` for infinity.*
